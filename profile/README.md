CN2 GIA这个词，在介绍里经常出现，但到底是什么意思，很多人并不清楚。

简单说：中国电信的骨干网分两种，CN2 GT（单程优化，去程普通线路，回程CN2）和CN2 GIA（双程都走CN2精品线路）。GIA是"Global Internet Access"的缩写，是电信自己的高端产品，理论上有带宽优先保障，拥堵时候GIA线路比普通线路更稳。
